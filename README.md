-**Proper 3rd person camera for Crimson Desert is a curated camera retune designed to give the game a lower, more grounded, and more cinematic third-person feel inspired by the camera style of Red Dead Redemption 2.**

-**The goal of this mod is not to add a large preset pack or completely redesign every camera mode in the game. Instead, it focuses on one polished setup that improves the default gameplay camera while keeping the game's built-in camera distance switching intact.**

-**This release aims to make the camera feel better during normal gameplay by lowering the view, softening the stock movement response, improving horse camera behavior, and slightly widening the near / first-person-style view. The result is a steadier and more immersive third-person camera that still feels natural inside Crimson Desert.**

-**Unlike a broader camera overhaul, this is a single curated release built around one clear goal: a proper third-person camera with a strong RDR2-style bias.**
